# Urban-Infrastructure-Failure-Risk
A predictive models that identify whether city infrastructure will fail

Project Overview
This project, developed for the Spring 2026 Machine Learning Module, aims to build a predictive system for a national urban resilience agency. 
The objective is to identify whether localized city infrastructure (power, water, transport) is likely to fail (1) or remain stable (0) during a specific monitoring window.

Note: This analysis uses a fictional dataset (urbandata.csv) representing city blocks under various environmental and operational stresses.


Tech Stack
Language: Python 3.11

Libraries: Pandas, Scikit-Learn, Matplotlib, Seaborn

Models: Logistic Regression, Decision Trees, Neural Networks (MLP)
